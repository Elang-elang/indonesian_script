# Pernyataan
setiap ada program pasti ada pernyataan, berikut adalah contoh penulisan kode dari beberapa pernyataan:

## Pernyataan Blok dan Non-Blok
pernyataan blok adalah pernyataan yang dibuka dengan "{" dan ditutup dengan "}". contoh:
```is
{
    teks nama = "anda!";
}

tuliskan format("{nama} {enter}", nama=nama, enter=enter);
// maka akan galat (error)
```
